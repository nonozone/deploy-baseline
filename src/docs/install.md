# Install

For local development of this repository:

1. Run `make build-skill`
2. Run `make install-local`

This installs the packaged skill into the local Codex skills directory.

The install target is:

- `$CODEX_HOME/skills/deploy-baseline-kit`
- or `~/.codex/skills/deploy-baseline-kit` when `CODEX_HOME` is not set
