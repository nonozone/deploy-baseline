# Product Overview

`deploy-baseline-kit` is an installable skill product for standardizing real projects toward the deploy baseline.

Primary use case:

- inspect an existing repository
- explain the gap from the baseline
- converge the repository toward the standard structure

Secondary use case:

- bootstrap a near-empty repository with the same baseline

The product is intentionally opinionated:

- baseline-first
- one main confirmation gate
- explicit `exceptions`
- offline-installable package output
