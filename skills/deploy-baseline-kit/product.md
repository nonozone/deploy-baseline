# Deploy Baseline Kit

`deploy-baseline-kit` is a skill product for standardizing real projects toward the deploy baseline.

Primary job:

- inspect an existing project
- explain the gap from the baseline
- converge the project toward the baseline with one main confirmation gate

Secondary job:

- bootstrap a near-empty project with the same baseline

Product stance:

- baseline-first, not compatibility-first
- explicit `exceptions`, not silent drift
- single-source packaging, not long-term duplicate template maintenance
